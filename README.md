# AIC & BIC Model Comparison Tool

A Python utility for comparing cosmological models using the **Akaike Information Criterion (AIC)** and the **Bayesian Information Criterion (BIC)**, based on Maximum Likelihood estimates extracted from MCMC output files.

---

## Overview

This script automates the statistical comparison of two models by:

1. Parsing the output of `getLikeStats()` to extract the best-fit Maximum Log-Likelihood value
2. Computing AIC and BIC for each model
3. Evaluating the evidence in favor of one model over the other using **Jeffreys' Scale**
4. Saving a formatted comparison report to a text file

---

## Formulae

$$\text{AIC} = 2k - 2\ln(\mathcal{L})$$

$$\text{BIC} = k\ln(N) - 2\ln(\mathcal{L})$$

Where:
- $k$ = number of free parameters of the model
- $N$ = total number of data points
- $\ln(\mathcal{L})$ = Maximum Log-Likelihood

---

## Jeffreys' Scale

Model preference is assessed from the absolute difference $|\Delta \text{AIC}|$ or $|\Delta \text{BIC}|$:

| $|\Delta|$ | Interpretation |
|---|---|
| $< 2$ | Weak / Inconclusive evidence |
| $2 – 6$ | Positive evidence |
| $6 – 10$ | Strong evidence |
| $> 10$ | Very strong (decisive) evidence |

## Requirements

- Python 3.x
- NumPy

Install dependencies with:
```bash
pip install numpy
```

---

## Input File Format

The script expects a plain-text file generated by `getLikeStats()`, containing a line of the form:

```
Best fit sample -log(Like) = 1328.142200
```

The value is the **negative** log-likelihood; the script automatically converts it.

---

## Usage

Edit the `__main__` block in `AIC_&_BIC.py` to configure your models and dataset:

```python
N_data_points = 2509   # Total number of data points

model_1 = {
    'name': 'LCDM',
    'path': '/path/to/LCDM/Like_statistics.txt',
    'k': 6   # Number of free parameters
}

model_2 = {
    'name': 'TPM',
    'path': '/path/to/TPM/Like_stats.txt',
    'k': 10
}

output_file = "/path/to/output/AIC_BIC_model_comparison.txt"

perform_model_comparison(model_1, model_2, N_data_points, output_file)
```

Then run:

```bash
python "AIC_&_BIC.py"
```

```
--------------------------------------------------------------------

--- DIFFERENCES (Model 2 - Model 1) ---
Delta AIC: -8.14
Delta BIC: 15.42

--- CONCLUSIONS (Jeffreys' Scale) ---
AIC Analysis: Strong evidence favoring TPM over LCDM (|Delta AIC| = 8.14)
BIC Analysis: Very strong (decisive) evidence favoring LCDM over TPM (|Delta BIC| = 15.42)
==========================================================
```

## Functions

| Function | Description |
|---|---|
| `calculate_information_criteria(path, k, N)` | Parses the likelihood file and returns AIC, BIC, and Max ln(L) |
| `get_evidence_statement(delta, criterion, model1, model2)` | Returns a Jeffreys' Scale verdict string |
| `perform_model_comparison(model1, model2, N, output)` | Orchestrates the full comparison and writes the report |

## Example Dataset

The default example compares the **ΛCDM** model ($k=6$) against the **TPM** model ($k=10$) on a combined dataset of $N=2509$ points from:
- Planck (TT,TE,EE + lowl)
- Planck-ACT Lensing
- DESI DR2 BAO
- DESY5 Dovekie SNe Ia
